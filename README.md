# Snake
## What is this?
This is classic "snake" game written in python using pygame library  
## Requirements
To play, you need the following things:
 - [python 3.5 or higher](https://www.python.org/downloads/)
 - pygame library 
```bash
pip install pygame
```
## Usage
 * Run main.py 
 ```bash
 python main.py
 ```
 * Enjoy!
